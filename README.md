## Setup Instructions

1. **Clone the repository**:
   ```bash
   git clone https://github.com/islamwell/unified-chat-archiver-extension.git
   ```

2. **Navigate to the project directory**:
   ```bash
   cd unified-chat-archiver-extension
   ```

3. **Install dependencies**:
   ```bash
   npm install
   ```

4. **Run the application**:
   ```bash
   npm start
   ```

## Features
- Archive and manage chat logs from various platforms.
- Responsive dashboard for easy navigation.
- Supports exporting archives in different formats.
- User-friendly interface with customizable settings.

## Usage Guide
After setting up the project, you can use the Chat Archive Dashboard extension by following these steps:

1. Launch the application using the command above.
2. Once the dashboard is open, connect your chat platforms by clicking on **Connect**.
3. Browse through your archives, and utilize the search feature to find specific chats.
4. Export your chat logs by selecting the desired format from the export menu.
5. Explore additional settings to customize your experience.

For detailed documentation, refer to the Wiki section in this repository.
